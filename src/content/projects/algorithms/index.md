An interactive web app client that visualises common sorting and pathfinding algorithms.

Includes the following sorting algorithms:

- Bubble sort
- Insertion sort
- Selection sort
- Merge sort
- Quick sort

Includes the following pathfinding algorithms:

- Dijkstra's shortest path
- A\* search
